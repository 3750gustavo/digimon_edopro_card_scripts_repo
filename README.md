# Edopro Digimon Cyber Sleuth Custom Card Scripts

![Game Search Screen for the keyword "Digimon"](https://files.catbox.moe/7bsr1l.JPG)

## Description
This repository contains scripts the 158 custom cards I made for Project Ignis Edopro based off the Digimon Cyber Sleuth video game.

## Installation
To install the cards into your copy of edopro, simply download the folder and extract it into your edopro expansions folder. The path should look something like this:

`ProjectIgnis\expansions`

## Usage
Once installed, you can search for the keyword "Digimon" in the game to find all of the cards.
